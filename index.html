Welcome to Bello

Bello is a very simple code formatting utility written in Ruby. It is released under the MIT License.

Why "Bello"?

Bello is Italian for beauty. So that's the reason Bello was picked as the name of the project.

Languages Supported

Bello currently supports

  • Matlab
